Prework Study Guide

HTML

  • HTML elements build the structure of a webpage and makes navigating easy
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The head element contains the metadats for the web application
  • Metadata is data about our webpage that the browser uses to serve our webpage to the user

CSS

  • "cascading style sheets"
  • used to describe the presentation of a website created with HTML
  • used to describe how each element on a webpage is rendered
  • a margin indicates how much space we want around the outside of an element
  • a padding indicates how much space we want around the content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • free open source version control system
  • allows developers to collab on projects without overwriting each other's work
  • is kinda confusing but its ok
